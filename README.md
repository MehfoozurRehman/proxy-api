# Proxy API

A simple Express-based proxy API server. This project allows you to forward requests, handle CORS, and manage API calls securely from your frontend applications.

## Features

- Express.js server
- CORS support
- Environment variable management with dotenv
- Easy to extend for custom proxy logic

## Tech Stack

- Node.js
- Express
- Axios
- CORS
- Dotenv

## Getting Started

### Development

```bash
npm install
npm start
```

The server will start on the default port (set in your environment variables or `index.js`).
